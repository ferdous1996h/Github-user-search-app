## Overview

This is a Github-user-search-app from Frontend Mentor...

### The challenge

Users should be able to:

- Search for GitHub users by their username
- See relevant user information based on their search
- Switch between light and dark themes View the optimal layout for the app
  depending on their device's screen size
- See hover states for all interactive elements on the page

Bonus: Have the correct color scheme chosen for them based on their computer
preferences. Hint: Research prefers-color-scheme in CSS.

### What I learned

- how to fetch data from 3rd party api
- Using useRef to switch theme
- useRef doesn't works like state and should use by function or conditioning.
- create custom date format
- how to make fetch request using export function and return from it.
- how to return data and catch error from **_Octokit_**
- how to watch and access error object using
  **_Object.getOwnPropertyDescriptors()_** and get error message.
- how to use use custom **'react-loader-spinner'** component for loading.
